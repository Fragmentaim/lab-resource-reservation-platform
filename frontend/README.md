# 前端说明

前端是实验室资源预约系统的完整 Vue 3 页面，使用 Vite 启动，默认端口为 `5175`。

## 启动

```powershell
cd D:\lab-booking-course\frontend
npm install
npm run dev
```

访问地址：

```text
http://127.0.0.1:5175
```

前端通过 Vite 代理访问后端接口：

```text
/api -> http://localhost:8081
```

## 页面模块

- 登录 / 注册
- 资源列表
- 资源详情和可预约时段
- 我的预约
- 我的通知
- 个人主页
- 管理端总览
- 用户管理
- 资源管理
- 时段管理
- 预约管理
- 字典管理
